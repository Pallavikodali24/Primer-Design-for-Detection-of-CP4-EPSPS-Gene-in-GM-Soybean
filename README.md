# Primer Design for Detection of the CP4 EPSPS Gene in Genetically Modified Soybean

## Project Overview
This project focuses on designing PCR primers to detect the presence of the CP4 EPSPS gene, a commonly used transgene in Roundup Ready genetically modified soybean.

## Objective
To design and evaluate specific primer pairs for the amplification of the CP4 EPSPS gene using bioinformatics tools and standard primer selection criteria.

## Background
The CP4 EPSPS gene confers glyphosate resistance in genetically modified crops. Detecting this gene is important for GMO identification, research, and regulatory purposes.

## Tools Used
- NCBI Nucleotide Database
- NCBI Primer-BLAST
- Primer3 Algorithm
- BLAST Specificity Analysis

## Methodology

### Step 1: Gene Sequence Retrieval
- Retrieved CP4 EPSPS gene sequence from NCBI.
- Accession Number: JF445290.

### Step 2: Primer Design
Parameters used:
- Product size: 100–300 bp
- Primer length: 18–24 bases
- Melting Temperature: 58–62°C
- GC Content: 40–60%
- Maximum Tm difference: <2°C

### Step 3: Primer Evaluation
Primer pairs were evaluated based on:
- Product size
- Self-complementarity
- 3' complementarity
- Melting temperature
- GC content
- Secondary structure formation

### Step 4: Final Primer Selection
Primer Pair 7 was selected as the best primer pair due to:
- Balanced GC content
- Suitable melting temperature
- Optimal product size
- Low self-complementarity
- Minimal chances of primer dimer formation
